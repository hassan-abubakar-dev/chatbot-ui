# Chatbot UI

A simple React-based chatbot UI with basic conversation logic using the `supersimpledev` Chatbot library.  
A responsive frontend chatbot app built with React, featuring real-time messaging simulation and environment-variable-based axios configuration for LAN or production use.

## Features
- Auto-scroll to latest message
- Simple AI-like response system
- Timestamped chat messages
- Clean responsive UI
- Environment-variable-based axios for LAN or production
- Mobile-friendly and fully responsive
